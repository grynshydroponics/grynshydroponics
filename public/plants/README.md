# Plant images (WebP)

Put one WebP image per plant here. **Filenames must match the `icon_img` field in `plantLibraryMaster.json`** (e.g. no spaces: `bellpepper.webp`, `cherrytomato.webp`).

Examples:

- `basil.webp`
- `bellpepper.webp`
- `cherrytomato.webp`
- `greenonion.webp`
- `romainelettuce.webp`
- `swisschard.webp`
- `mustardgreens.webp`
- `bushbeans.webp`

The app loads these for the plant dropdown/list icons. Set `"icon_img": "bellpepper.webp"` (etc.) in the master JSON for each plant.
